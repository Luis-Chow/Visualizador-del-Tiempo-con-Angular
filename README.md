# App Visualizador de Tiempo

Proyecto hecho en Angular para mostrar la hora de 11 formas distintas.

## Stack tecnico
- Angular (Standalone)
- RxJS para el flujo del tiempo
- LocalStorage para usuarios

## Sistema de Login
El componente de Auth maneja registro y entrada en una sola pantalla. 
Guarda los datos en el navegador para que no se borren al recargar.

## Como correrlo
1. Instalar dependencias: `npm install`
2. Correr el server: `ng serve`
3. Abrir en localhost:4200
